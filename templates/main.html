The task

The description of the task is described here https://github.com/Code-Maven/wis-advanced-python-2021-2022 but because some of you may not have access to this directory, find the instructions below.

Assignments

The followig 3 assignments are all preparations to create the 4th assignment. I think the best would be to create one after the other and reuse the code from the earlier steps later on, but if you go straight to step 4 that's also fine by me. You can jump right to the 4th assignment and do that.

You can store the solutions in the same git repository where you put the HTML pages in the previous assignment or in a totally new repository. Do NOT copy the data files as they might contain information (e.g. phone numbers, email addresses) that other student do not want to show publicly. Your code can assume that the files are in a directory next to your code that you have the following layout of directories on your computer:

course/
  wis-advanced-python-2021-2022/ # the clone of this repo
  application/ # the repo where your application is.

List all the people on the command line

Write a command line script that receives a path to a directory (the students directory of this repository) and can list the name of all the people after reading the JSON files.

List people on the web

  • Build a page that will list all the people in abc order.
  • Each name is a link.
  • When clicking on the link display all the details of the student.

Command line search engine

Improve the first script so in addition to the path it will also receive a search term. It should print the names of the student only of any of the data fields match the given text.

Student search engine

  • Build a search engine that allows you to type in some text and you will list the names of all the students from the students/ directory of this repository that match the criteria in any of their fields.
  • Sort the list in abc order.
  • Turn each name into a link and when you click on a name, show all the details of the person.
  • If it is a URL, convert it into a link.
